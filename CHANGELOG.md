# Changelog

All notable changes to the MegaLinter Configuration extension will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.0.1] - 2026-01-01

### Added
- Initial release of MegaLinter Configuration extension
- Visual form-based configuration editor for `.mega-linter.yml` files
- React-powered WebView interface
- Automatic JSON Schema fetching from MegaLinter repository
- Field validation based on MegaLinter's official schema
- YAML parsing and serialization
- Command palette integration: "MegaLinter: Open Configuration"
- Context menu integration for `.mega-linter.yml` files
- Automatic workspace detection for existing configuration files
- Support for creating new configuration files
- Save/load functionality with user feedback
- Comprehensive documentation and user guide

### Features
- Dynamic form generation from JSON Schema
- Real-time field validation
- Support for nested objects and arrays
- Type-appropriate input controls (text, number, checkbox, etc.)
- VS Code theme integration
- Error display and validation messages
